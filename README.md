I'm working through the [Blind 75](https://neetcode.io/practice) — the canonical list of DSA problems every serious engineer should know cold.

**Language of choice: Go.**

Previous submissions exist in C++, but going forward everything gets solved (or re-solved) in Go. If a problem was already done in C++, it's getting a Go pass too. No exceptions.

Why Go?
